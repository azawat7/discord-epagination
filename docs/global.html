OtherButtonsOptions

Options for the other buttons (delete, first, last).

Properties:
Name Type Description
enabled boolean

Whether the button should be enabled or not.
position number

Position of the button in the row. You need to set a positive index, the default buttons (back, foward) are in the array and you can't change their positio. Don't forget that the buttons are receiveing position in this order : first -> last -> delete, so make sure to set a correct index if you want to do something very specific.
